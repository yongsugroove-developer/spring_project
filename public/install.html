Mobile Access

설치 안내

현재 공개 배포는 Android 서명 APK와 iPhone Safari PWA를 대상으로 합니다. 로그인과 실제 데이터 사용에는 네트워크 연결이 필요합니다.

Android APK

  1. 운영자가 전달한 APK 다운로드 링크를 Android 기기에서 엽니다.
  2. 브라우저 또는 파일 앱에서 APK를 내려받습니다.
  3. 설치 경고가 나오면 해당 앱에 대해 알 수 없는 앱 설치를 허용합니다.
  4. 설치가 끝나면 앱을 실행하고 회원가입 또는 로그인을 진행합니다.
  5. 새 버전을 받으면 기존 앱 위에 다시 설치해 업데이트합니다.

iPhone Safari PWA

  1. iPhone에서 Safari로 배포 도메인을 엽니다.
  2. 하단 또는 상단의 공유 버튼을 누릅니다.
  3. 홈 화면에 추가를 선택합니다.
  4. 생성된 홈 화면 아이콘으로 앱처럼 실행합니다.
  5. 첫 실행 후 회원가입 또는 로그인을 진행합니다.

안내 사항

  • 공개 베타에서는 개별 회원가입을 사용합니다.
  • 현재 결제 기능은 placeholder 상태이므로 앱에서 노출하지 않습니다.
  • 오프라인에서는 설치된 셸만 열릴 수 있고 실제 플래너 데이터 편집은 네트워크가 필요합니다.
